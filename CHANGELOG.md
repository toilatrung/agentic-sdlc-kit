# Changelog

All notable framework changes are recorded in this file. The format follows Keep a Changelog principles, and releases use Semantic Versioning.

## [1.0.0] - 2026-06-22

### Added

- Repository-native governance, planning, execution, reporting, retrieval, and intelligence models.
- Twelve reusable governance, planning, execution, and report templates.
- Task-driven context retrieval and bounded context-package support.
- Project onboarding workflow for SRS, contract, and architecture inputs.
- Agent prompt pack for bootstrap through release management.
- Standard-library framework validator and v1.0.0 release audit assets.

### Changed

- Packaged the internal framework skeleton as the reusable Agentic SDLC Kit distribution.

### Security

- Validation and operating policies prohibit secrets, fabricated evidence, unauthorized lifecycle transitions, and governance bypass.
